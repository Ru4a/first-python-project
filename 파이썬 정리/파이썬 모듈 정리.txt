파이썬 모듈(Module)

함수, 변수, 클래스들을 모아 놓은 파일
모듈은 다른 파이썬 프로그램에서 불러와서 사용할 수 있게 만든 파이썬 파일이다.

1. 메인 프로그램

2. 모듈

import (모듈명)
from (모듈명) import (함수나 클래스) / * : 전체 함수나 클래스를 불러온다

import로 모듈을 전체 불러왔을 때에는
모듈.함수 or 클래스()
from을 통해서 모듈명에서 한 함수나 클래스를 불러왔을 때에는


if __name__ = "__main__":
(내용)
모듈에서 프로그램을 실행했을 때 내용을 실행하고
모듈이 아닌 메인 프로그램에서 실행했을 때에는 실행하지 않는다.

같은 파일에서 모듈을 불러오는 것은 성공했지만
도저히 다른 파일에서 모듈을 불러오는 것이 안됨..
예시를 따라해보고 구글에 문제점을 쳐서 같은 문제를 가진 사람들의
해결법을 보고 따라해봤는데 실패했다.

다른 파일의 모듈을 그냥 복사해서 같은 파일에 저장하던가
